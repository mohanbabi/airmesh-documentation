Before Airmesh First Setup

Before starting the Airmesh setup, ensure the following:

  1. On your PC/laptop, register on Tailscale and obtain an authorization key.
  2. Save the key securely in a notepad or a known location, as the auth key will not be shown again.

Note: The authorization key will be required during the activation step.

First Setup Airmesh

  1. Follow the connection diagram to connect the components.
  2. Turn on your laptop/PC.
  3. Power up the circuit using a 5-volt power supply through the servo cable.
  4. Wait for the Airmesh Wi-Fi hotspot/access point to appear.
  5. Once the Airmesh Wi-Fi is available, connect to the network.
  6. Open a browser and type http://10.0.2.100:5500, then press Enter.
  7. You will now see the Tailscale User Interface (UI).
  8. Click the "Connect to Tailscale" button in the UI.
  9. When prompted, enter your saved pre-auth key in the "Enter your pre-Auth Key" input box.
  10. Click the button below the input box to proceed.
  11. Once successful, you will see a success message.
  12. Go to the Tailscale admin window to verify if the new device has been added.
  13. If added, the device can now directly communicate with other devices in the Tailscale network.

Note: If you encounter any issues during the VPN setup, please contact us.

A video reference for all the above steps is available. Please refer to it for further assistance.

Flight Control Setting

  1. Connect to the Airmesh Wi-Fi network.
  2. Open a browser and navigate to http://10.0.2.100:3000.
  3. Click on the "Flight Controllers" option on the left side of the page.
  4. The Flight Controller setting page will open.
  5. Follow the video reference provided to complete the setup.

Ground Control Stations

  1. Ensure your Ground Control Station PC or mobile device is connected to the Tailscale network.
  2. Follow the video reference to control the drone using the cellular service.
  3. If you encounter any issues, please contact us.
